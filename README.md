# SkiDay with ASP.NET and React.js 

Demo project for ASP.NET 5 and React.js 

## Installation

- Open src\SkiDay.WebApp\ClientUI in Command Prompt and start 'webpack'
- Open src\SkiDay.sln in VS2015, build, run
