# SkiDay with ASP.NET MVC and React.js 

Demo project for ASP.NET MVC 5 + EF6 + React.js + REDUX  

## Installation

- Install node
- Open src\SkiDay.WebApp\ClientUI in Command Prompt 

    `npm install`

    `npm start`

- Open src\SkiDay.sln in VS2015, build, run
